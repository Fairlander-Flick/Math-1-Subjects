### Question 1: Short Answers & Rapid Tests

- [ ] **Complex Number Operations:** Executing division, conjugation ($\bar{z}$), and exponentiation ($z^n$) efficiently.
    
- [ ] **Parametric Cross Product:** Computing $u \times v$ where vectors contain variables (e.g., $\lambda$).
    
- [ ] **Polynomial Substitutions:** Solving high-degree polynomials like $p(x) = x^{10} - 6x^5 + 9$ by substituting $y = x^5$.
    
- [ ] **Eigenvalue Inspection:** Directly identifying $\lambda_i$ from the main diagonal of Triangular or Diagonal matrices without using Laplace expansion.
    
- [ ] **Invertibility Analysis:** Determining the range of a parameter (e.g., $\beta$) such that $\det(A) \neq 0$.
    

### Question 2: Formal Proofs

- [ ] **Complex Mathematical Induction:** Proving summation formulas $\sum_{k=1}^n f(k)$ that include the imaginary unit $i$ through step-by-step induction.
    
- [ ] **Multiplicity & Double Roots:** Proving $x = x_0$ is a double root by verifying $p(x_0) = 0$ and $p'(x_0) = 0$ (or checking the remainder of division) to solve for unknown coefficients $\alpha, \beta$.
    

### Question 3: The "Golden Hit" (Least Squares)

- [ ] **Summation-to-Matrix Transformation:** Converting a complex error function given in sigma notation ($\sum$) into the standard matrix norm form: $\|Ax - b\|_2^2$.
    
- [ ] **Solving Normal Equations:** Deriving the optimal vector $x$ by solving the system $A^*Ax = A^*b$.
    

### Question 4: Comprehensive Matrix Theory

- [ ] **Structural Analysis of $A$:**
    
    - [ ] Calculating the **Determinant** ($\det(A)$).
        
    - [ ] Finding the **Rank** ($\text{rank}(A)$).
        
    - [ ] Computing the **Kernel** (Null Space) dimension and finding its **Basis**.
        
- [ ] **Eigenvector Verification:** Confirming if a specific vector $v$ is an eigenvector by performing the multiplication $Av$ and checking if it equals $\lambda v$.
    
- [ ] **Definiteness Proof:** Demonstrating that a matrix is **Positive Definite** (PD) by verifying that all its eigenvalues are strictly positive ($\lambda_i > 0$).
